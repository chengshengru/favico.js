favico.js

favico.js v0.3.9

在你的 favicon 上使用徽章、图片或视频

查看 GitHub

尝试：

支持 Chrome、Firefox、Opera 和 IE11+

徽章

为你的 favicon 和徽章添加动画。你可以自定义动画类型、位置、背景颜色和文字颜色。

滑动动画

var favicon=new Favico({
    animation:'slide'
});
favicon.badge(1);

渐入渐出动画

var favicon=new Favico({
    animation:'fade'
});
favicon.badge(2);

弹出动画

var favicon=new Favico({
    animation:'pop'
});
favicon.badge(3);

弹出并渐入渐出动画

var favicon=new Favico({
    animation:'popFade'
});
favicon.badge(1);

无动画

var favicon=new Favico({
    animation:'none'
});
favicon.badge(2);

位置设置

var favicon=new Favico({
    position : 'up'
});
favicon.badge(3);

形状设置

var favicon=new Favico({
    type : 'rectangle',
    animation: 'slide',
});
favicon.badge(1);

支持自定义字体

var favicon=new Favico({
    fontFamily : 'FontAwesome',
    elementId : 'badgeFont'
});
favicon.badge('\f0a2');

颜色设置

var favicon=new Favico({
    bgColor : '#5CB85C',
    textColor : '#ff0',
});
favicon.badge(3);

图片 / 视频 / 摄像头

从图片、视频甚至是摄像头流创建 icon

图片 icon

var favicon=new Favico();
var image=document.getElementById('imageId');
favicon.image(image);

HTML5 视频 icon

var favicon=new Favico();
var video=document.getElementById('videoId');
favicon.video(video);
//stop
favicon.video('stop');

摄像头视频 icon

只是为了好玩但是它可以工作 :)
支持 Chrome、Firefox 和 Opera 

var favicon=new Favico();
favicon.webcam();
//stop
favicon.webcam('stop');

徽章选项

属性 默认值
bgColor #d00 徽章背景颜色
textColor #fff 徽章文字颜色
fontFamily sans-serif 字体（Arial、Verdana、Times New Roman、serif、sans-serif 等）
fontStyle bold 字体风格（normal、italic、oblique、bold、bolder、lighter、100、200、300、400、500、600、700、800、900）
type circle 徽章形状（circle、rectangle）
position down 徽章位置（up、down、left、upleft）
animation slide 徽章动画类型（slide、fade、pop、popFade、none）
elementId false 图像元素 ID
element false DOM 元素的位置
dataUrl false 调用每一个动画的 URI 参数数据的方法
